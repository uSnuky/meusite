Laboratório

Experimentos interativos e demonstrações técnicas

Sistema de Partículas Interativo

Este experimento demonstra a criação de um sistema de partículas interativo usando JavaScript puro e Canvas. As partículas respondem ao movimento do cursor e podem ser configuradas através dos controles.

Tecnologias utilizadas: Canvas API, JavaScript, Matemática vetorial

Mini Editor de Desenho

Este experimento demonstra a criação de um editor de desenho simples usando a API Canvas do HTML5. Você pode desenhar livremente, criar formas geométricas e personalizar cores e tamanhos.

Tecnologias utilizadas: Canvas API, JavaScript, Eventos de Mouse

Sintetizador Web Audio

Este experimento demonstra o uso da Web Audio API para criar um sintetizador musical interativo com visualização em tempo real. Toque no teclado virtual ou use seu teclado do computador (teclas A-L) para tocar notas.

Tecnologias utilizadas: Web Audio API, Canvas para visualização, JavaScript

Motor de Física 2D

Este experimento demonstra a implementação de um motor de física 2D simples usando JavaScript puro. Você pode adicionar objetos, ajustar parâmetros físicos e observar as interações entre os objetos.

Tecnologias utilizadas: JavaScript, Canvas, Física computacional
